在主库中添加测试测试数据后，在从从库中也会同步；
如果主库关闭，从库会充当主库；
此时如果原主库再启动，则变为从库；

Q&A

1.在从库中无法查看数据库中的数据报 code: 13435

error: { “$err” : “not master and slaveOk=false”, “code” : 13435 }
注意看错误信息： 不是主机，slaveOk=false

解决方案： 进入mongo控制台
输入

rs.slaveOk()

2.No route to host

解决方案： 更新防火墙规则

$ firewall-cmd –reload
